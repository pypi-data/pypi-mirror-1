-----------------------------
export content in file system
-----------------------------

    >>> self.portal
    <PloneSite ...
    >>> self.loginAsPortalOwner()
    >>> from Testing import makerequest
    >>> self.portal = makerequest.makerequest(self.portal)

Install collective.synchro in portal::

    >>> context = self.portal
    >>> from collective.synchro import config
    >>> config.HAS_FSS
    True
    >>> self.install_portal_synchronisation()
    True
    >>> self.install_fss()
    True


create an document with file to be exported::

    >>> self.create_fss_content('fss')
    >>> self.portal.folderfss.fss
    <DummyContent at /plone/folderfss/fss>
    >>> self.portal.folderfss.fss.getFile()
    <VirtualFile at /plone/folderfss/fss/file>


    >>> self.portal.portal_synchronisation.getPlugins()
    [...collective.synchro.plugins.fss.FssExporter...]
    >>> from collective.synchro.data import SynchroData
    >>> data = SynchroData(self.portal.folderfss.fss)

We delete content::

    >>> self.portal.folderfss.manage_delObjects('fss')

And reimport data in portal::
    >>> data(self.portal)

Verify that the file in File System Storage is correctly installed::

    >>> self.portal.folderfss.fss.getFile().getData()
    'text'

Delete all objects created for this test::

    >>> self.portal.manage_delObjects('folderfss')

And uninstall collective synchro::

    >>> self.uninstall_portal_synchronisation()
    True
    >>> self.uninstall_fss()
    True
