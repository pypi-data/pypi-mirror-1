quintagroup.plonetabs Package Readme
=========================

Overview
--------

The product allows Plone websites to manage portal_actions tool
  from portal_control_panel configlet.

This is a fully rewritten version of qPloneTabs product. It is based
on kss javascript framework and plone 3 technologies.

  * Plone 3.0+

Tested on:

  * Firefox

  * IE 7
