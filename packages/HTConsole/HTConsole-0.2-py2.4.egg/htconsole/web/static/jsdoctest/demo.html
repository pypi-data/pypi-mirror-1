JS Doctest Demo

This is a demo of JS Doctest. Doctests are <pre> elements with the class doctest, like: 
<pre class="doctest">
  $ print(
  >   'hi');
  hi
</pre>
Note that $ is used for a normal line, and > is used for a continuation line.
Examples Output
$ writeln('hi')
hi
$ 1+10
11
$ function foo() {
>   for (var i=0; i<3; i++) {
>     writeln(i);
>   }
> }
$ foo()
0
1
2
$ 'anything ' + 'else?'
anything else?
