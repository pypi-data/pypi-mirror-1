easyshop.taxes Package Readme
=========================

Overview
--------

Tax management for EasyShop

  - Management of any sum of taxes
  - Assignment of taxes on base of criteria (http://pypi.python.org/pypi/easyshop.criteria): category, group, product, customer, country, date
