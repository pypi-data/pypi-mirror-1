  Аннотация:

    Продукт предоставляет виджет для поля повторения другого поля.

  Идея реализации:

    Виджет наследуется от zope.app.form.browser.widget.SimpleInputWidget, 
    перекрывая метод getInputValue.
    
    Параметры:
    
      checkfield -- name поля для сравнения
    
    Частный случай:
    
      PasswordRepeatWidget(PasswordWidget, RepeatWidget)
    
    Способ использования:
    
      <widget
        field="passwordRepeat"
        class="ks.widget.repeatwidget.PasswordRepeatWidget"
        checkfield="REG.password"
      />
    
    где checkfield -- это действительное название поля в форме
    (в общем случае отличается от имени поля в интерфейсе) для 
    сравнения.
