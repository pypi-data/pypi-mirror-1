===================
medialog.fullnamefield
===================

-------
Purpose
-------

medialog.fullnamefield adds a new Archetypes field for storing fullname fields.

What makes it special?

*  It autofills the fullname field with the fullname of the currently authenticated member.

How do I use it?

See the example Archetype content type created in:
medialog/fullnamefield/contents/fullname.py
