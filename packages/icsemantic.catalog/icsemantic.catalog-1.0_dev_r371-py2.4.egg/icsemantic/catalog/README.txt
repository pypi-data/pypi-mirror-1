icSemantic.Catalog searches Plone contents using thesaurus relation terms.
Terms are related by related concepts, equivalent terms and translated terms,
borrowed and narrowed terms. They can be used to relate users and documents
by keywords or searchable texts. These terms are used to extend the portal
catalog in order to gather a broader result set from the catalog.

Functionality

    * Extends catalog functions with three new indexes associated to the thesaurus
    * Searches documents using related, equivalent and translated terms
    * New features in advanced search
    * Smart folder with these new search indexes.
