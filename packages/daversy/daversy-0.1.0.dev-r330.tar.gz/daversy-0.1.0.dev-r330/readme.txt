PyDvs is an attempt to rewrite Daversy in Python for shorter implementation and simpler maintenance.

You'll need the following tools to run it:

1. Python 2.4.3 (from http://www.python.org/2.4.3/)
2. cx_Oracle    (from http://starship.python.net/crew/atuining/cx_Oracle/index.html)
3. lxml         (from http://codespeak.net/lxml/)
