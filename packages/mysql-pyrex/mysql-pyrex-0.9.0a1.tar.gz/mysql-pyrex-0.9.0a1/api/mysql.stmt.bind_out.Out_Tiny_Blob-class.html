TINYBLOB output binding.

The constructor takes the following arguments:

  • who: The object on which to set the value.
  • what: The attribute name to set on the object.
  • max_size: The size of the buffer in bytes to use to fetch data. Defaults to 4000.
  • store_stream: If True, will store a stream object (Output_Stream) which has a read method. Otherwise it stores the entire result as a string. Defaults to False.

Note that Out_Tiny_Text is an alias for this object.
