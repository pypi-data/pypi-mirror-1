append(self, val, typ=None, space=True, keepS=False, indent=False)

source code 

Appends val. Adds a single S after each token except as follows:

  • typ COMMENT

    uses cssText depending on self.ser.prefs.keepComments

  • typ cssutils.css.CSSRule.UNKNOWN_RULE

    uses cssText

  • typ STRING

    escapes ser._string

  • typ S

    ignored except keepS=True

  • typ URI

    calls ser_uri

  • val {

    adds

after
  • val ;

    removes S before and adds

after
  • val , :

    removes S before

  • val + > ~

    encloses in prefs.selectorCombinatorSpacer

  • some other vals

    add *spacer except space=False
