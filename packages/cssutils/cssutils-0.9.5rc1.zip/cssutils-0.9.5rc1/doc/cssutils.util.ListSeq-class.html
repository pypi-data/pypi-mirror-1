(EXPERIMENTAL) A base class used for list classes like css.SelectorList or stylesheets.MediaList

adds list like behaviour running on inhering class' property seq

  • item in x => bool
  • len(x) => integer
  • get, set and del x[i]
  • for item in x
  • append(item)

some methods must be overwritten in inheriting class
