Welcome to armsim’s documentation!¶

armsim is a project which aims to simulate an AT91SAM9260 processor, to do so it needs the ARMv5TE instruction set (it should also support Thumb and Jazelle instructions later)

You can find a page of resources, where all the documentation links should be found
