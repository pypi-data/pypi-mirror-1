============
Introduction
============
With inqbus.plone.panorama you can easy include panorama images in Kupu and
TinyMCE.

How to use inqbus.plone.panorama?
----------------------------------------------------------------------

1. install inqbus.plone.panorama
2. with Kupu & co edit a page and add an panorama image
3. mark the panorama image and set Panorama as style
4. after saving the page you should have an animated panorama in your page
5. you can change some settings under
/portal_properties/inqbus_plone_panorama_properties in the ZMI

See it here in action in inqbus.plone.panorama demo_:
.. _demo: http://www.inqbus-hosting.de/open-source-software/inqbus.plone.panorama/
