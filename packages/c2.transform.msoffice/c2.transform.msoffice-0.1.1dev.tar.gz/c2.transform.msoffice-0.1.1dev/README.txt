Introduction
============

Description
-------------

This product enables to search MS-Offile files on Plone.

A files
--------
- MS-Offile files

  - MS-Word 95-2000, XP, 2007
  - MS-Excle 95-2000, XP, 2007
  - MS-PowerPoint 95-2000, XP, 2007

Requirement
-------------

- Plone

  - Plone 3.0-3.3 (tested by 3.3.2 on CentOS 5.3 64bit)
  - May be able to use on Plone 4 (but not tested)

- Java

  - JRE 1.6 (tested by 1.6.0)
  - May be able to use JRE 1.5 (but not tested)
