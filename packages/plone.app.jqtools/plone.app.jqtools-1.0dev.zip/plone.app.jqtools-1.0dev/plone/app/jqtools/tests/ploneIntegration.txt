Introduction
------------

Setup the test framework::

    >>> from zope.component import getMultiAdapter
    >>> from Products.Five.testbrowser import Browser
    >>> browser = Browser()
    >>> portal_url = 'http://nohost/plone'

We should already be installed::

    >>> import plone.app.jqtools


Javascript Resource and View
----------------------------

We should have one items in the JS registry::

    >>> jsreg = portal.portal_javascripts
    >>> ids = jsreg.getResourceIds()
    >>> '++resource++plone.app.jqtools.js' in ids
    True

Open the main JS code item as a resource::

    >>> browser.open(portal_url+'/++resource++plone.app.jqtools.js')

And, make sure it's got our code in it::

    >>> print browser.contents
    /*
     * jquery.tools ... - The missing UI library
     * 
    ...
