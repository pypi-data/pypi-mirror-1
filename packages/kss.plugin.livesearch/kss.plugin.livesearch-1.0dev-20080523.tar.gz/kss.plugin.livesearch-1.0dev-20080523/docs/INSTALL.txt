kss.plugin.livesearch Installation
==================================

 * When you're reading this you have probably already run 
   ``easy_install kss.plugin.livesearch``. Find out how to install setuptools
   (and EasyInstall) here:
   http://peak.telecommunity.com/DevCenter/EasyInstall

 * Copy the file called ``kss/plugin/livesearch/kss.plugin.livesearch-configure.zcml`` 
    in the
   ``/path/to/instance/etc/package-includes`` directory.
